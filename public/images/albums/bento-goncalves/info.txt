Este é um arquivo de informações para o álbum de Bento Gonçalves.

Para adicionar suas fotos reais:
1. Baixe as imagens do Google Photos manualmente
2. Coloque-as nesta pasta
3. Certifique-se de que os nomes dos arquivos correspondam aos nomes usados no arquivo de dados

Recomendações:
- Use imagens otimizadas para web (formato JPG ou WebP)
- Mantenha um tamanho razoável (recomendado: largura máxima de 1920px)
- Use nomes de arquivos descritivos (ex: vinícola-aurora-01.jpg)
